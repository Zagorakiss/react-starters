# isomorphic-react-spa-ts

Works stable on Node.js v6.10.0.
# Installing
Clone the project and install dependencies
```
git clone {REPO_NAME}
sudo npm install
```

Launching in dev mode with hot reload

```
npm run start
```
Launching in prod mode

```
npm run start:prod
```

Analyze code for potential errors
```
npm run lint
```
